# NIP rent

Aplikacja do rozliczania najmowanych mieszkań. Program pozwala wczytywać dane z plików JSON dotyczące mieszkań, pokoi, najemców, rachunków oraz przelewów, a następnie analizować i wyświetlać informacje o rozliczeniach najmu.

## Funkcjonalność

Aplikacja umożliwia:
- wczytywanie danych mieszkań z pliku JSON
- wczytywanie danych najemców
- wczytywanie rachunków
- wczytywanie przelewów
- analizę rozliczeń między najemcami a rachunkami
- proste wyświetlanie danych w konsoli
